# 每日心情 v0.1.0 发布说明

这是每日心情日记应用的首个版本，为用户提供了跨平台的日记记录体验。

## 主要功能
- 支持纯文本和Markdown格式的日记
- 多种视图：日历视图、列表视图和收藏视图
- 图片附件功能：拍照或从相册选择
- WebDAV云同步功能，支持在不同设备间同步日记
- WebDAV媒体管理功能，用于管理已上传的图片
- 标签系统和心情记录：记录每天的情绪变化

## 问题修复
- 修复Windows平台上WebDAV配置无法保存的问题
- 改进WebDAV连接测试，实现真正的连接验证
- 修复布尔值在SQLite中的存储问题
- 修复WebDAV客户端库的空安全和类型转换问题

## 安装说明
### Windows
1. 下载发布包中的`DiaryApp-v0.1.0-Windows.zip`文件
2. 解压到您想要的位置
3. 运行`diary_app.exe`
